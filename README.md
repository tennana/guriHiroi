# ぐり。拾い
第41回奈良高専祭 情報処理研究会展示作品として作りました。古く難しいこともしていない作品ですが、未だに[公開している](http://www.vector.co.jp/soft/win95/game/se442783.html )ぐらいなのでソースコードも公開して良いやと思いました。

#### ゲーム説明(当時)
←→キーでカゴを動かしてひたすら上から落ちてくる 
「どんぐり。」を集めつつ3分間生き延びるゲーム。 

1回3分で、簡単なランキング機能を搭載しています。 
暇潰しにどうぞ。 

#### 注意点
Windows7で起動したら×ボタンで終了できないアプリケーションになっていました。ESCキーで離脱できます。

#### 意図
- 教室にマシンを持ち込んでブースにする形での展示だったので、1回のゲームに時間がかからないこと。
- 対象層は高専生以外も多く、別にプログラミング技巧に興味はないだろうと思われるのでシンプルに作成。

#### 結果
- 割合遊ばれたので気の抜けた音楽がよく響いていた。
- Hard Modeで突撃してくるアレのインパクトで笑いも取れた。
- 「ぼくはにゅぅ。とか取らないんや」と謎のこだわりを見せる少年に遭遇。
- 面白がらせるのに複雑さは必須要件じゃあないんだなとようやく実感できた。
